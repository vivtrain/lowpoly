# PROJECT:lowpoly

(work in progress)

This project uses an NVIDIA Jetson TX1 with OpenCV4Tegra and CUDA for image processing. The goal is to generate a low-poly-artwork-style video given some source video using edge detection, object tracking, and pseudorandom graph/network generation.
